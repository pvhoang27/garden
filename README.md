# Garden CMS

## Mô tả

Dự án web quản lý cây cảnh với giao diện tối giản, tập trung vào hình ảnh, hỗ trợ responsive, lazy loading ảnh/video, có phần quản trị nội dung (admin CMS) cho CRUD cây cảnh và danh mục. Sử dụng React cho front-end và Node.js/Express cho back-end.

## Cấu trúc chính

- Front-end: React (client)
- Back-end: Node.js/Express (server)
- Quản trị: Đăng nhập, quản lý danh mục, quản lý cây cảnh (CRUD, upload ảnh, video, Rich Text Editor)
- Tối ưu: Responsive, lazy loading, tối giản, dễ dùng

## Hướng dẫn khởi động

1. Cài đặt dependencies cho client và server
2. Chạy server: `npm start` trong thư mục server
3. Chạy client: `npm start` trong thư mục client

## Triển khai

- Có thể deploy lên các dịch vụ như Vercel, Heroku, hoặc server riêng
- Hỗ trợ gắn tên miền riêng (ví dụ: mygarden.net)
